# Assets de marca

Coloca aqui los archivos finales de marca para usar en la web.

## Logo recomendado

- `logo-cesar-arturo-horizontal.png`: version completa con simbolo + texto.
- `logo-cesar-arturo-mark.png`: solo el monograma CA dentro del cuadro.
- `logo-cesar-arturo-mark.svg`: version vectorial del monograma, ideal para favicon y web.
- `favicon.svg`: icono final para navegador.
- `og-image.png`: imagen para compartir en redes.

## Exportaciones recomendadas

- PNG transparente: 1200 px de ancho para el logo horizontal.
- PNG transparente: 512 x 512 px para el monograma.
- SVG para la version final si se redibuja en vector.
- Mantener una version clara y una version oscura si el logo se usa sobre fondos distintos.

## Nota sobre el logo actual

El logo generado con IA funciona bien como direccion visual: monograma CA, marco cuadrado y acabado cobre. Para hacerlo mas profesional, conviene vectorizarlo o redibujarlo en una herramienta de diseno para limpiar bordes, equilibrar proporciones y exportar versiones transparentes.
